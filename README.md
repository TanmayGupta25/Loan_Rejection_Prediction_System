# Loan Rejection Prediction System

This is a production-ready Machine Learning application built with Streamlit and Scikit-Learn.

## Structure
- `app.py`: Main Streamlit application.
- `models/`: Contains the binary Pipeline and feature schema.
- `deployment/`: JSON metadata and configuration.

## Deployment
Push this folder to a GitHub repository and connect it to [Streamlit Community Cloud](https://share.streamlit.io/).
